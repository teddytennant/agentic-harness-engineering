Create a Python module at `/app/even_sum.py` that defines a function:

```python
def even_sum(nums):
    ...
```

`even_sum` takes an iterable of integers and returns the sum of only the even
numbers. Requirements:

- An empty input returns `0`.
- Negative even numbers count (e.g. `-4` is even).
- It must not mutate the input.

Only the file `/app/even_sum.py` is required; no other output.
